# weather_in_app
一个用flutter写的查看天气的App，你可以选择想要查看的城市，或者可以根据当前的IP来获取所在城市的天气信息。  
你可以从 https://www.tianqiapi.com/  这个网站来注册一个属于你自己的控制台。  
这里用到的是免费七天的天气接口。

城市列表使用的是网站提供的城市json文件。
